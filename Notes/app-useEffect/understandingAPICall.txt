    2. SHIMMER UI

        1. what is shimmer UI?

            --> Its a Fake UI that appears on the page when the api is taking
                some time to load.

            --> fake UI means some boxes or skeleton that appers before the page loads.

            --> Shimmer UI is a loading placeholder technique used in apps and websites to improve perceived performance while content is being fetched.

            --> Instead of showing a blank screen or a spinning loader,
            shimmer UI displays a skeleton layout of the
            page — grayed-out placeholder shapes (rectangles, circles, lines)
            that mimic the structure of the actual content. A shimmering/glowing
            animation sweeps across these shapes, giving the impression that
            content is "loading in."

            --> Common examples:

                Facebook/LinkedIn feed cards loading
                YouTube video thumbnails before they appear
                Product cards on e-commerce sites

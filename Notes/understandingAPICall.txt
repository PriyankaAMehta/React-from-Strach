1. structure :

json
 └── data
      └── cards[4]
           └── card
                └── card
                     └── gridElements
                          └── infoWithStyle
                               └── restaurants  ← this is the ARRAY you need
                                    └── [0]
                                         └── info
                                              └── name: "Pizza Hut"

2. So to get the restaurants array:
        
        json.data.cards[4].card.card.gridElements.infoWithStyle.restaurants


3. THE FLOW 

    1. STEP 1: Page Loads → Body.js renders first time

        const [ListOfRestaurants, setListOfRestaurants] = useState(resList);
        //                                                          ↑
        //                                            starts with mock/empty data
        
        * React renders the page with mock data first, while API loads in background.

    2. STEP 2: useEffect fires AFTER page loads

            useEffect(() => {
                console.log("useEffect called");
                fetchData();   // ← calls the API
            }, []);
            //  ↑
            //  empty array means "run this ONCE, after first render"


    Think of useEffect like:
                "Hey React, AFTER you paint the page,
                THEN go fetch the real data"

    3. STEP 3: fetchData() hits the API

        const fetchData = async () => {

        // STEP 3a: Go to this URL and get the response
        const data = await fetch("https://www.swiggy.com/...");
        //            ↑
        //         "await" means WAIT here until API responds
        //         don't move to next line until data comes back


        // STEP 3b: Convert response to JSON (readable format)
        const json = await data.json();
        //            ↑
        //         again WAIT until conversion is done


        // STEP 3c: Drill into json to find the restaurants array
        console.log(json) // you can see the whole response here
    };

    4. STEP 4: Drill into the JSON to find restaurants

    json                              → whole API response
    json.data                         → data inside it
    json.data.cards                   → array of 12 cards
    json.data.cards[4]                → 4th card (restaurant grid)
    json.data.cards[4].card.card      → go deeper
    ...gridElements                   → go deeper
    ...infoWithStyle                  → go deeper
    ...restaurants                    → ✅ ARRAY OF 8 RESTAURANTS

    const restaurants = 
    json.data.cards[4].card.card.gridElements.infoWithStyle.restaurants

            // restaurants = [
            //   { info: { name: "Pizza Hut", avgRating: 4.2, ... } },
            //   { info: { name: "KFC",       avgRating: 4.5, ... } },
            //   ... 8 total
            // ]

    5. STEP 5: Save the array into State

            setListOfRestaurants(restaurants);
        //  ↑
        //  this UPDATES the state
        //  React sees the state changed
        //  React AUTOMATICALLY re-renders the page with new data


    Think of state like a **whiteboard**:

    First render:   whiteboard has mock data  → page shows mock data
    API responds:   whiteboard gets updated   → React repaints page with real data

    6. STEP 6: React re-renders Body.js with real data
        return (
            <div className="res-container">

                {ListOfRestaurants.map((restaurant) => (
                //  ↑
                //  Now ListOfRestaurants has 8 real restaurants
                //  map() loops 8 times

                    <RestaurantCard 
                        key={restaurant.info.id}  // unique id for each card
                        resData={restaurant}       // pack data into bag
                    />
                ))}

            </div>
        )

    7. STEP 7: Each RestaurantCard receives its data and displays

       // Loop 1: restaurant = Pizza Hut data
        // Body sends: <RestaurantCard resData={Pizza Hut data} />

        const RestaurantCard = (props) => {
            
            // Open the bag
            const { resData } = props;
            // resData = { info: { name: "Pizza Hut", ... } }

            // Take out what you need
            const { name, avgRatingString, cuisines, costForTwo } = resData.info;
            const { deliveryTime } = resData.info.sla;

            // Paint it on screen
            return (
                <div className="res-card">
                    <h3>{name}</h3>              {/* Pizza Hut */}
                    <h4>{cuisines}</h4>          {/* Pizzas */}
                    <h4>{avgRatingString}</h4>   {/* 4.2 */}
                    <h4>{costForTwo}</h4>        {/* ₹300 for two */}
                    <h4>{deliveryTime} mins</h4> {/* 32 minutes */}
                </div>
            )
        }

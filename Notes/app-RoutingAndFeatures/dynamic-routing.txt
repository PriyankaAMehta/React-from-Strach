1. Yes, you can change it — but it must match whatever you named the parameter in your route definition.

        The name comes from the route, e.g. in App.js:

                <Route path="/restaurant/:resId" element={<RestaurantMenu />} />

        The :resId part defines the name. So useParams() must destructure with the same name:

                const { resId } = useParams(); // ✅ matches :resId

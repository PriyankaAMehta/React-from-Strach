1.  What is functional componenet --> its normal JS function.

2.  What is class component --> its JS Class.

3.  JavaScript Class Syntax :

    1. Use the keyword class to create a class.

    2. Always add a method named constructor():

    3. class ClassName {
       constructor() { ... }
       }

4.  How to write class component in react?

    1. class Component_Name extends React.Component{

    }

    2. extends React.Component tells react that it is a component.

    3. It has a render() method returns the JSX. This JSX is displayed on the screen.

    4. How are props passed in class?

    --> props are passed by the constructor method of the class

    5. Example of normal class in JS

       class Car {
       constructor(name, year) {
       this.name = name;
       this.year = year;
       }
       }

       1. The example above creates a class named "Car".

       2. The class has two initial properties: "name" and "year".

       3. A JavaScript class is not an object. It is a template for JavaScript objects.

5.  Life Cyle Of Class based components :

        example :

            class UserClass extends React.Component {
            constructor(props) {
                super(props);

                this.state = {
                count: 0,
                count2: 2,
                };
            }

            render() {
                const { name } = this.props;
                const { count } = this.state;
                return (
                <div className="user-card">
                    <h2>Name : {name}</h2>
                    <h3>Location : Hanau</h3>
                    <h4>Contact : @mehta_pri </h4>
                    <h2>Count = {count}</h2>

                    <button
                    onClick={() => {
                        this.setState({
                        count: this.state.count + 1,
                        });
                    }}
                    >
                    Click Me
                    </button>
                </div>
                );
            }
            }

            export default UserClass;

    1. Whenever the above component is rendered, The constructor () method is called first and then render () is called.
